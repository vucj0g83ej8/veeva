# LINE LIFF UI/UX 設計圖

此資料夾存放 LINE LIFF 版本的 UI/UX 設計材料。

## 資料夾

- `reference/`: 參考風格與設計靈感。
- `wireframe/`: 低保真線框圖。
- `mockup/`: 高保真 UI mockup。
- `screenshot/`: 實作後的畫面截圖。
- `assets/`: 圖片、icon、既有截圖素材。

## 主要文件

- `liff-ui-template.html`: LINE LIFF 顧客端 UI/UX 模板，可直接用瀏覽器開啟預覽。
- `app-screens-guide.html`: 既有畫面說明文件。
- `app-screens-guide.pdf`: 既有畫面說明 PDF。
