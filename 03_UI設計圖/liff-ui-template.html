顧客端核心畫面

下方四個畫面定義 LIFF 會員系統的主要風格：首頁任務、最新資訊、兌換券、會員與邀請好友。實際開發時可以沿用這些元件和間距。

元件規則

每個新 LIFF 專案都可以先套用這些元件規則，再依照品牌調整色彩與內容。

Bottom Navigation

最多 4 個入口，顧客端建議固定為活動、資訊、兌換、會員。選取狀態使用薄荷綠膠囊底色。

Primary CTA

高度 46px 到 48px，實心深綠或 LINE 綠。每張卡片只保留一個主要操作。

System Messages

使用右上通知鈴開啟小型浮層。訊息不全螢幕化，避免打斷任務流程。

Reward Card

兌換券卡片只放商品名稱、期限與操作按鈕，降低掃描負擔。

Referral Flow

推薦流程使用三步驟說明，主要按鈕為 LINE 分享，降低使用者理解成本。

Review Status

審核中、已通過、已發券必須用一致的 badge 樣式呈現，避免同狀態不同顏色。
