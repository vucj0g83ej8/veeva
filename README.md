# LINE LIFF 會員系統版本資料

此資料夾為 LINE LIFF + Firebase Hosting / Firestore 版本的規劃與改版基礎資料，獨立於原本交付資料夾之外。

## 01_程式碼

目前 Flutter 專案的 LIFF 改版基礎。後續可在此版本上改為 Firebase Hosting + LIFF Web App。

## 02_規劃書

LINE LIFF + Firebase 版本規劃書，依新規則分成：

- `project/`：初始 LIFF + Firebase 專案規劃與 baseline。
- `update/`：後續錯誤修正、調整、優化與驗收回饋。
- `feature/`：未來新增功能與第二階段需求。

`project/` 包含：

- LIFF 會員系統架構
- Firebase Hosting / Firestore / Cloud Functions 規劃
- 推薦分享與帳號綁定流程
- Firestore 資料結構
- Admin 後台與 UI/UX 規劃
- 驗收查驗清單與發佈檢查清單

## 03_UI設計圖

目前顧客端與後台畫面說明書、PDF、HTML 與截圖素材，可作為 LIFF 版 UI 改版參考。

## 04_報價單

目前正式報價單 HTML 與 PDF，可作為 LIFF 版本報價調整基礎。
