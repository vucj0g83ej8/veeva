# 開發工時估算

## 估算假設

- 以 LINE LIFF + Firebase Hosting + Firestore 為第一版架構。
- 使用既有 Flutter Web 原型作為改版基礎。
- Admin 後台沿用目前畫面方向並調整資料串接。
- Veeva 表單可於 LIFF 中開啟，若不支援則改外開。

## 階段總覽

| 階段 | 工作內容 | 估算工時 |
| --- | --- | ---: |
| Phase 0 | 需求確認、技術確認、資料結構規劃 | 10 |
| Phase 1 | Flutter 顧客端 App 開發 | 60 |
| Phase 2 | Admin 後台 App 開發 | 52 |
| Phase 3 | 後端 API、資料庫與管理邏輯 | 82 |
| Phase 4 | 第三方整合與通知流程 | 26 |
| Phase 5 | 測試、修正、部署與上線準備 | 30 |
| Phase 6 | 文件、操作手冊與交付 | 6 |
| **合計** |  | **266 小時** |

## 備註

本版報價以總工時 266 小時估算，其中 Phase 0 與 Phase 6 小計不計費，計費工時為 250 小時；以每小時 NT$ 400 計算，未稅金額為 NT$ 100,000。
